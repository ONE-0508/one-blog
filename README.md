## 个人博客项目说明

本仓库用于搭建一个**个人博客网站**，包含前台阅读端与后续可扩展的后台管理端。当前阶段主要聚焦于**前端部分**，后端服务会在后续迭代中补充与接入。

---

## 前端概览

- **技术栈**：
  - 框架：React 19（函数组件 + Hooks）
  - 构建：Vite
  - 语言：TypeScript
  - 样式：TailwindCSS 4
- **主要能力（对应 PRD）**：
  - 博客文章：首页列表、文章详情、分类/标签、归档等。
  - 笔记记录：笔记列表/详情（碎片化内容展示）。
  - 留言板块：读者留言与展示。
  - 作品展示：项目/作品卡片与详情页。
  - 数据统计：基础访问与内容统计展示（视后端实现情况）。
  - 移动端适配：移动端优先布局、明暗主题切换。

详细的前端架构与页面设计，可参考：

- `frontend/docs/前端架构设计文档.md`
- `frontend/docs/前端页面设计文档.md`

## 目录结构（前端相关）

```bash
frontend/
  ├─ src/
  │  ├─ pages/        # 各业务页面（首页、详情页等）
  │  ├─ components/   # 通用 UI 组件
  │  ├─ layouts/      # 布局组件（如 AppLayout）
  │  └─ ...           # 其他工具、样式等
  └─ docs/            # 前端相关设计文档
```

## 本地开发

确保已安装 pnpm（推荐）或 npm/yarn，这里以 pnpm 为例：

```bash
cd frontend
pnpm install      # 安装依赖
pnpm run dev      # 启动本地开发服务
```

启动后按终端提示访问本地地址（通常为 `http://localhost:5173`）。
